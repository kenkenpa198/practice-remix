### 01-installation

```shell
# Build
npx remix vite:build

# Run
npx remix-serve build/server/index.js
```

### 02-bring-your-own-server

```shell
# Run by Express
node server.js

# Run by Express with debug mode
node --inspect server.js
```

### 03-development-workflow

```shell
# Start dev server
npm run dev
```
